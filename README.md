# 役員報酬・会社員比較シミュレーター

会社員として働く場合と、法人化して役員報酬・節税施策（iDeCo+・小規模企業共済・社宅概算・出張旅費概算）を使う場合の **手取り / 実質手取り / 法人残キャッシュ / 個人＋法人合計 / 各税負担 / 会社員との差額** を概算比較する Web シミュレーターです。最適化モードでは役員報酬・賞与を総当たりして合計キャッシュ上位 20 パターンを表示します。

## 技術構成

Next.js (App Router) / TypeScript / Tailwind CSS / React Hook Form / Zod / Recharts / decimal.js / Vitest

## コマンド

```bash
npm run dev      # 開発サーバ (http://localhost:3000)
npm test         # ユニットテスト (Vitest)
npm run build    # 本番ビルド
```

主要画面:

- `/` — トップ（説明・各モードへの導線）
- `/simulator` — 入力フォーム（左）＋ 結果カード・比較表・グラフ・最適化（右）。入力変更でリアルタイム再計算。
- `/simulator/result` — 結果詳細（比較表＋会社員/法人の負担内訳）

## 料率・税率マスタの差し替え

税制・社会保険料率は年度改定されるため、年度マスタを単一ソースに集約しています。

- 年度マスタ: `src/lib/constants/rateMaster/2026.ts`
  - 社会保険料率（健保 9.85% / 子育て支援金 0.23% / 介護 1.62% / 厚年 18.3%、協会けんぽ東京 2026）
  - 標準報酬月額表は **等級区分（固定）× 料率で実行時生成**。料率を 1 か所変えれば全等級が再計算されます。
  - 所得税ブラケット・給与所得控除・基礎控除・住民税率/均等割
- 年度解決: `src/lib/constants/rateMaster/index.ts` の `getRateMaster(year)`
- **新年度を追加するには**: `2027.ts` 等を作成して `RateMaster` 型を満たし、`index.ts` の `REGISTRY` に登録します。

## 計算ロジック

`src/lib/calc/*` は UI 非依存の純粋関数で、すべて Vitest でテストしています（`npm test`）。

- `socialInsurance` / `bonusSocialInsurance` — 月給は標準報酬月額、賞与は標準賞与額。厚年 月 150 万・健保 年累計 573 万上限、賞与年 4 回以上は報酬扱い。
- `salaryIncome` / `incomeTax` / `residentTax` — 給与所得控除・所得控除・超過累進・復興特別所得税・住民税所得割 10%＋均等割。
- `idecoPlus` — 会社負担分は法人損金、個人負担分は小規模企業共済等掛金控除。社保削減効果は 0。合計月額 5,000〜62,000 円（1,000 円単位）でバリデーション（令和7改正で 23,000→62,000 円・2026年12月1日施行。上限は `rateMaster/2026.ts` の `idecoPlus.monthlyMax`）。
- `taxSaving` — 社宅・出張旅費・小規模企業共済・経営セーフティ共済。
- `corporateTax` — 役員報酬支給前利益から各損金を控除し実効税率で課税。
- `simulator` / `optimizer` — 会社員/法人ケースの比較と最適報酬探索。

## 検証済みの数値（令和8年・2026年）

一次情報（国税庁・協会けんぽ）と突き合わせ済みです。

- 協会けんぽ東京 令和8年度の料率（健保9.85%／子ども・子育て支援金0.23%／介護1.62%／厚年18.3%）・標準報酬月額表 全50等級・折半端数処理：**公式と完全一致**（出典: 協会けんぽ R8 東京 保険料額表）。
- 給与所得控除（最低保障74万・収入220万以下）・基礎控除（令和8年分 104/104/104/67/62万）：**令和8年度税制改正の確定値**に更新済み（出典: 国税庁「令和8年4月 源泉所得税の改正のあらまし」）。
- 所得税の超過累進税率・復興特別所得税2.1%（2026年）・住民税 所得割10%・住民税基礎控除43万・均等割5,000円：**2026年計算として正しい**ことを確認。

## 既知の前提・簡略化

- **iDeCo+**：従業員向けの中小事業主掛金納付制度のため、従業員ゼロの一人社長（役員のみ）が利用できるかは公式に不明確（グレー）。一人社長は「企業型DC」が一般的。税務上の計算（会社分=損金／個人分=所得控除／社保削減ゼロ）は正しい。
- **会社負担の社会保険料**：本人と同額として概算。事業主のみ負担の「子ども・子育て拠出金（約0.36%・厚年ベース）」は未計上のため、会社負担をわずかに過小評価（本人手取りには影響なし）。
- **扶養控除**：一般区分（所得税38万・住民税33万）で概算。特定扶養（19〜22歳, 住民税45万）等の加算は未対応。
- **基礎控除の判定**：給与所得のみを前提（給与所得＝合計所得）。副業・年金・配当等の他の所得がある場合は要拡張。
- **厚生年金の賞与上限**：本来は「同月合算後に月150万円」。本ツールは支給回ごとに判定（支給月が分かれる前提なら実害なし）。
- **住民税の均等割**：非課税判定（低所得者）を行わず一律課税。
- **令和10年分以後**：基礎控除・給与所得控除が物価スライドで再改定されるため、年度マスタ（`2027.ts` 等）の追加が必要。

## 注意

本ツールの数値はすべて **概算** です。料率・税率改定時は `src/lib/constants/rateMaster/` を差し替えてください。各施策の実行前に税理士・社労士等の専門家へ確認してください。
